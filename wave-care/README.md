# Wave Care - Frontend

Aplicação web desenvolvida com Next.js.

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## Pré-requisitos

- Node.js 20+
- npm
- Git

---

## 1. Clonar o projeto

git clone https://github.com/isatech07/wave-care.git

cd wave-care

---

## 2. Instalar as dependências

npm install

---

## 3. Configurar o arquivo .env.local

Crie um arquivo chamado:

.env.local

Exemplo:

NEXT_PUBLIC_API_URL=http://localhost:3001

---

## 4. Executar o projeto

npm run dev

Aplicação disponível em:

http://localhost:3000

---

## Estrutura

app/

components/

public/

styles/

---

## Observações

- O backend deve estar em execução.
- Certifique-se de que a variável `NEXT_PUBLIC_API_URL` aponta para a URL correta da API.
